## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone https://github.com/nfarrux2197/clother_shop_BETA.git
cd clother_shop_BETA
```

### 2. Создать виртуальное окружение

```bash
python -m venv .venv
.venv\Scripts\activate      # Windows
source .venv/bin/activate   # Mac/Linux
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Создать файл `.env`

```
SECRET_KEY=твой-секретный-ключ
DEBUG=True
```

### 5. Выполнить миграции

```bash
py manage.py migrate
```

### 6. Создать суперпользователя

```bash
py manage.py createsuperuser
```

### 7. Запустить сервер

```bash
py manage.py runserver
```

Открыть в браузере: [http://127.0.0.1:8000](http://127.0.0.1:8000)
Админка: [http://127.0.0.1:8000/admin](http://127.0.0.1:8000/admin)

## 🔧 Как это работает

### Запрос пользователя
```
Браузер → urls.py → views.py → models.py → БД
                                         ↓
Браузер ← HTML шаблон ← views.py ← данные
```

### Модели (models.py)
- **Category** — категория товаров. Содержит название, slug (уникальный URL) и изображение
- **Product** — товар. Привязан к категории через `ForeignKey`. Содержит название, описание, цену, старую цену, изображение и статусы (`is_new`, `is_available`)
- `discount_percent` — вычисляемое свойство, автоматически считает процент скидки из `price` и `old_price`

### URL маршруты (urls.py)
| URL | View | Описание |
|---|---|---|
| `/` | `home` | Главная страница |
| `/category/<slug>/` | `category_detail` | Страница категории |
| `/product/<slug>/` | `product_detail` | Страница товара |
| `/admin/` | Django Admin | Панель управления |

### Шаблоны (templates/)
- `base.html` — базовый шаблон с навбаром, футером, подключением CSS и JS
- `clothers_shop/home.html` — главная страница, наследуется от `base.html` через `{% extends %}`

### Статические файлы (static/)
- `css/main.css` — весь дизайн сайта, кастомные CSS переменные, адаптивность
- `js/main.js` — интерактивность: фильтры, кнопка корзины, автоскрытие сообщений

### Админка
Все модели зарегистрированы в `admin.py`. Slug заполняется автоматически из названия (`prepopulated_fields`). Можно редактировать цену и статусы прямо в списке товаров (`list_editable`).
